## 📝 Basic Editing

- `i` → Insert mode before cursor
- `a` → Append after cursor
- `o` → Open new line below and insert
- `O` → Open new line above and insert

## 📦 Saving & Quitting

- `:q!` → Quit without saving

## 🚀 Navigation

- `h` / `l` → Left / Right  
- `0` → Beginning of line  
- `^` → First non-blank char of line  
- `$` → End of line  
- `w` → Next word start  
- `e` → End of word  
- `b` → Back one word  
- `gg` → Start of file  
- `G` → End of file  
- `:n` → Go to line `n` (e.g. `:42`)
